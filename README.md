# En el a)

- Inclusión o subtipo
  En los constructores de las subclases de Viaje usando super, lo cual se encarga de que la subclase pueda usar los atributos de la superclase.
  Un ejemplo de código:
    "public ViajeAltaVelocidad(int capMaximaPasajeros, double distancia, int cantidadEstaciones) {
        super(capMaximaPasajeros, distancia, cantidadEstaciones);
    }"

 -  Paramétrico
   No lo usamos ya que no implementamos ni listas ni arrays en nuestro código

 - Coerción
   Tampoco lo utilizamos

 - Sobrecarga
   Tampoco lo utilizamos, pero si usamos sobrescritura
